SVG Mesh Polyfill

This code allows SVG 2.0 style mesh gradients to be rendered in web environments that don't natively support it.

To use, include mesh-polyfill.js just before the closing tag in your file, and make sure it is in the same directory as your file.

For HTML: <script src="mesh-polyfill.js"></script>

For SVG: <script type="text/javascript" xlink:href="mesh-polyfill.js"></script>
